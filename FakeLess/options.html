Klucz API Google Gemini

Wymagany do wykrywania deepfake w wideorozmowach i analizy obrazów AI.

Klucz jest bezpiecznie przechowywany w przeglądarce.

Jak uzyskać klucz API (darmowy):

  1. Wejdź na aistudio.google.com/app/apikey
  2. Zaloguj się na swoje konto Google
  3. Kliknij "Create API Key" lub "Utwórz klucz API"
  4. Wybierz projekt (lub utwórz nowy)
  5. Skopiuj wygenerowany klucz (zaczyna się od AIza...)
  6. Wklej klucz w pole powyżej i kliknij "Zapisz ustawienia"
Uwaga: Klucz API pozwala na analizę obrazów przez Gemini AI. Bez klucza rozszerzenie działa w trybie lokalnym (mniej dokładnym).
